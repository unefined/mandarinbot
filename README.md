# MANDARIN 
Features:

-  🔧 Config with bindings function
-  📟 Open source code


Functions:

- 🍊 Picking oranges              
- 🍊 Add new point oranges         
- 🔮 Enable/Disable script         
- 🔐 Fastening your seat belt           
- 🚘 Autopilot (Press W)             
- 🍒 Multi purchase (Shop 24/7)    
- 🧳 Trunk Load/Clicker

# Installation

1. git clone https://github.com/unefined/mandarinbot.git
2. cd mandarinbot
3. npm i
4. node (ru or en)
